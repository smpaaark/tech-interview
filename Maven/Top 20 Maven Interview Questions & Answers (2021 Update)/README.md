## 메이븐(Maven)
Maven은 프로젝트 관리 도구입니다.    
개발자에게 완전한 빌드 생명 주기 프레임워크를 제공합니다.    
Maven 명령을 실행할 때 Maven에서 POM 파일을 찾습니다.    
POM에 설명된 리소스에서 명령을 실행합니다.

## Maven이 관리하는 영역
메이븐은 개발자의 다음 작업을 처리합니다.
* Build
* Documentation
* Reporting
* Dependencies
* SCMs
* Releases
* Distribution
* Mailing list

## Maven의 3가지 빌드 생명 주기
* Clean
  * 이전 빌드에 의해 만들어진 artifact를 정리합니다.
* Default(build)
  * 애플리케이션을 생성하는데 사용됩니다.
* Site
  * 프로젝트가 사이트 documentation를 생성합니다.
