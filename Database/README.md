## 데이터베이스란 무엇이라고 생각합니까?
데이터베이스는 데이터를 저장하고 목적에 맞게 구성하는 조직화된 데이터 모음입니다.   
* 예를 들어 도서관 관리자는 도서관에서 이용 가능한 책과 관련된 모든 정보의 데이터베이스를 관리합니다.
